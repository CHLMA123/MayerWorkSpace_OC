CSS盒子模型

在浏览器下，每一个html元素都会被解析为一个装有东西的盒子。盒子本身有自己的边框（border），盒子里的内容到盒子边框的距离称为填充（padding），盒子边框与其它盒子之间的距离为边界（margin）。 在css 模型设计中，元素真实的宽度和高度不仅仅由width和height来决定，还包括内边距、外边距、边框来组成。如下图所示：
